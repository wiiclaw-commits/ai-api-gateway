## 使用方法

### 在 Telegram 中发送消息

1. **简单任务** - 直接描述需求
   ```
   帮我创建一个用户登录 API
   ```

2. **指定 Agent** - 使用方括号前缀
   ```
   [前端] 创建一个登录页面
   [DevOps] 配置 Docker 部署
   ```

3. **复杂任务** - 描述完整需求
   ```
   开发一个电商平台，需要前端 Vue 界面、后端 Java API 和 Kubernetes 部署
   ```

### 桥接脚本

```bash
#!/bin/bash
# 使用示例
./telegram-delegate.sh "任务描述" "会话 ID" "用户 ID"
```
